# A股专家 - Skill文档

## 概述

本skill定义了与汤圆（主Agent）协作进行A股ETF周频交易的完整框架。

---

## 核心策略

### 交易框架
| 项目 | 设定 |
|-----|------|
| 本金 | 6万 |
| 仓位方案 | 方案B：3万×2只ETF |
| 交易频率 | 周频（周一买，周五卖） |
| 标的池 | 科技+材料领域ETF |

### 风控纪律
| 类型 | 规则 |
|-----|------|
| 止损 | -3%预警，-5%强制止损 |
| 止盈 | 5%+考虑减仓，8%清仓 |
| 仓位上限 | 单只不超过3万，同时最多2只 |
| 现金管理 | 满仓轮动，不留现金 |

## 每日协作流程

### 早上9点（交易日）
1. 用户发送持仓截图
2. 汤圆分析：
   - 持仓盈亏情况
   - 当日市场/板块动态
   - 是否需要调整
3. 讨论决策

### 周一（买入日）
- 开盘前确认2只标的
- 盘中执行买入
- 记录买入理由

### 周五（卖出日）
- 收盘前评估持仓
- 执行卖出纪律
- 记录卖出理由、盈亏

### 周末
- 复盘本周操作
- 选定下周候选标的
- 更新策略迭代

## 协作原则

1. **汤圆不预测市场** - 只分析、讨论、提醒纪律
2. **用户拥有最终决策权** - 买卖由用户执行
3. **严格记录** - 每笔交易必须记录，用于复盘迭代
4. **持续进化** - 根据实战经验调整策略
