# DFT-Sim 案例研究索引

本目录包含完整的实际计算案例，每个案例都提供详细的计算流程、结果分析和验证对比。

## 案例列表

### 1. [硅(Si)的完整第一性原理计算](si_complete_study.md)

**难度**: ⭐⭐ (入门级)  
**计算成本**: 低 (8核·小时)  
**适用软件**: VASP, QE

**涵盖内容**:
- 结构优化与k点收敛
- 能带结构与有效质量
- 态密度分析
- 声子性质与热容
- 光学性质 (PBE vs HSE06对比)
- EOS拟合与体模量
- 完整Python分析脚本

**学习价值**: 第一性原理计算的标准流程，适合初学者入门

### 2. [单层MoS₂的二维材料计算](mos2_2d_materials.md)

**难度**: ⭐⭐⭐ (进阶级)  
**计算成本**: 中 (24核·小时)  
**适用软件**: VASP, QE

**涵盖内容**:
- 2D材料特殊设置 (库仑截断/真空层)
- 自旋轨道耦合 (SOC)
- 应变工程与带隙调控
- 激子效应 (BSE计算)
- 异质结构建 (MoS₂/石墨烯)
- 能带对齐分析
- 谷极化效应

**学习价值**: 二维材料计算的特殊考虑，应变工程方法

---

### 3. [LiCoO₂锂离子电池正极材料](licoo2_battery.md)

**难度**: ⭐⭐⭐⭐ (高级)  
**计算成本**: 高 (100+核·小时)  
**适用软件**: VASP (推荐), QE

**涵盖内容**:
- DFT+U参数优化 (U值测试)
- 电压曲线计算 (不同Li含量)
- NEB离子扩散能垒
- 缺陷形成能与化学势
- 相稳定性分析
- 微观动力学模型
- 容量与循环稳定性预测

**学习价值**: 电池材料的系统计算方法，电化学性质预测

### 4. [Pt(111)表面CO氧化催化](pt_catalysis.md)

**难度**: ⭐⭐⭐⭐ (高级)  
**计算成本**: 高 (80+核·小时)  
**适用软件**: VASP (推荐)

**涵盖内容**:
- 表面模型构建 (slab模型)
- 吸附能与位点分析

- 覆盖度效应
- 微观动力学建模
- 火山曲线分析

**学习价值**: 催化计算的完整流程，微观动力学与DFT结合

### 5. [Bi₂Se₃拓扑绝缘体](bi2se3_topological.md)

**难度**: ⭐⭐⭐⭐⭐ (专家级)  
**计算成本**: 极高 (200+核·小时)  
**适用软件**: VASP + Wannier90 + Z2Pack

**涵盖内容**:
- 拓扑不变量 (Z₂) 计算 (宇称法/Wilson loop)
- 能带反转验证
- Slab模型表面态
- 狄拉克锥与费米速度
- 自旋-动量锁定
- 层投影能带分析
- 应变调控拓扑相变

**学习价值**: 拓扑材料计算的标准流程，SOC应用

### 6. [MAPbI₃钙钛矿太阳能电池](mapbi3_perovskite.md)

**计算成本**: 极高 (300+核·小时)  
**适用软件**: VASP

**涵盖内容**:
- 有机-无机杂化结构建模

- 缺陷容忍性分析
- 离子迁移 (NEB)
- AIMD有限温度模拟
- 合金化效应
- 压电/铁电性质

**学习价值**: 软晶格材料计算，光电器件材料设计

## 案例对比

| 案例 | 体系类型 | 核心方法 | 计算成本 | 典型产出 |
|------|----------|----------|----------|----------|
| Si | 3D半导体 | 基础DFT | ⭐⭐ | 能带、声子 |
| MoS₂ | 2D半导体 | SOC+BSE | ⭐⭐⭐ | 光学、应变效应 |
| LiCoO₂ | 电极材料 | DFT+U+NEB | ⭐⭐⭐⭐ | 电压、扩散 |
| Pt(111) | 金属表面 | 表面+NEB | ⭐⭐⭐⭐ | 催化活性 |
| Bi₂Se₃ | 拓扑绝缘体 | SOC+Z₂ | ⭐⭐⭐⭐⭐ | 表面态、狄拉克锥 |
| MAPbI₃ | 杂化钙钛矿 | AIMD+缺陷 | ⭐⭐⭐⭐⭐ | 离子迁移、相变 |
| Cu | 金属 | 输运+表面 | ⭐⭐⭐ | 电导率、费米面 |
| GaN | 宽禁带半导体 | 压电+掺杂 | ⭐⭐⭐⭐ | 带隙、异质结 |
| CO₂RR | 电催化 | CHE模型+吸附 | ⭐⭐⭐⭐ | 过电位、选择性 |
| 固态电解质 | 离子导体 | NEB+AIMD | ⭐⭐⭐⭐⭐ | 离子电导率 |
| 热电材料 | 热电转换 | BoltzWann+BTE | ⭐⭐⭐⭐⭐ | ZT值优化 |
