# 质量检查清单 (Quality Checklist)

> 每章完成后必须逐项检查
> 全部打勾才能算"完成"

---

## 📖 内容质量

### 费曼法检查
- [ ] 核心概念有生活化比喻
- [ ] 比喻能通过"8岁小孩测试"
- [ ] 直觉解释→形式化定义→应用实例的链条完整
- [ ] 避免生硬的术语堆砌

### 数学推导
- [ ] 所有公式从零推导，不跳步
- [ ] 每一步有"为什么"的解释
- [ ] 提供数值示例验证（小数据集演示）
- [ ] 关键公式用彩色标注或框选
- [ ] 推导流程图清晰

### 代码实现
- [ ] NumPy基础版本可运行
- [ ] PyTorch框架版本可运行
- [ ] 代码有详细注释
- [ ] 提供可视化工具
- [ ] 有完整可运行的示例脚本
- [ ] 代码风格统一（PEP8）

### 章节连贯性
- [ ] 与前一章有自然过渡
- [ ] 引用前面章节的成果
- [ ] 为后一章做铺垫
- [ ] 术语翻译统一

---

## 🎨 格式规范

### 文档格式
- [ ] 标题层级正确（#/##/###）
- [ ] 章节编号连续
- [ ] 代码块标注语言（```python）
- [ ] 数学公式使用LaTeX（$...$ 或 $$...$$）
- [ ] 图表有标题和说明文字
- [ ] 列表使用规范（- 或 1. 2. 3.）

### 引用规范
- [ ] 参考文献APA格式
- [ ] 文中引用标注作者年份
- [ ] 链接可点击访问
- [ ] 图片注明来源

---

## 🎯 教育价值

### 学习目标
- [ ] 章节开头明确学习目标
- [ ] 学完能做什么清楚说明
- [ ] 前置知识明确列出

### 学习支持
- [ ] 有常见错误提醒
- [ ] 有避坑指南
- [ ] 有扩展阅读建议
- [ ] 有相关章节交叉引用

### 练习设计
- [ ] 9道练习题（3基础+3进阶+3挑战）
- [ ] 基础题巩固概念
- [ ] 进阶题深入理解
- [ ] 挑战题培养创新
- [ ] 提供参考答案或思路

---

## 🔧 技术细节

### 代码质量
- [ ] 无语法错误
- [ ] 无运行时错误
- [ ] 边界情况处理
- [ ] 随机种子固定（可复现）
- [ ] 性能合理（不故意写慢代码）

### 数学准确性
- [ ] 公式符号统一
- [ ] 维度检查（矩阵乘法合法）
- [ ] 梯度计算正确
- [ ] 数值稳定性考虑

- [ ] 图表 ≥ 5个

- [ ] 关键公式完整推导
