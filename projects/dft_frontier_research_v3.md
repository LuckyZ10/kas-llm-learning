## 1. 实时光学吸收与阿秒电子动力学

### 1.1 实时TDDFT在阿秒科学中的应用 (npj Comp. Mater., 2025)

**研究背景**
2023年诺贝尔物理学奖授予阿秒光脉冲生成方法，开创了阿秒科学新纪元。实时TDDFT(rt-TDDFT)成为分析超快电子动力学的关键理论工具。

**核心应用**
- **阿秒瞬态吸收光谱(ATAS)**：追踪电子态演化
- **高次谐波产生(HHG)**：固体中相干辐射机制
- **时间分辨光电子能谱**：RABBIT测量分析
- **自旋动力学**：磁性材料超快退磁

**Co M₂,₃边ATAS研究案例**
```
实验观测：吸收边附近光吸收急剧下降
DFT解释：
  1. 激光激发驱逐多数自旋电子
  2. 少数自旋电子聚集于原子区域
  3. 局域自旋极化降低
  4. 交换劈裂与芯态蓝移
  5. 吸收边附近吸收减少
```

## 3. 缺陷化学DFT计算标准化

### 3.1 形成能计算完整流程

**中性缺陷**
```
E^f_0 = E_0 - E_p + Σn_iμ_i

其中：
  E_0: 含缺陷超胞总能量
  E_p: 完美超胞总能量
  n_i: 原子i数目(移除为负)
  μ_i: 原子i化学势
```

**带电缺陷**
```
E^f_q = E_q - E_p + E_corr - Σn_iμ_i + q(ε_F + ΔV_{0/p})

附加项：
  E_corr: 有限尺寸修正
  q: 缺陷电荷
  ε_F: 费米能级
  ΔV_{0/p}: 电势对齐项
```

### 3.2 有限尺寸修正方法

**超胞法**
- 多超胞尺寸计算外推
- 至少3种超胞尺寸
- 外推至无限大极限

**FNV修正** (Freysoldt-Neugebauer-Van de Walle)
```python
# 修正项组成
E_corr = E_lat + qΔV_{q/0}

E_lat: 点电荷晶格能
ΔV_{q/0}: 模型与DFT势差
```

**示例：金刚石空位**
| 超胞 | E(p) | E(0) | E(q) | ΔV_{0/p} | E_corr | E^f_q |
|------|------|------|------|----------|--------|-------|
| 2×2×2 | -627.68 | -609.58 | -620.57 | -0.097 | 1.84 | 13.33 |
| 3×3×3 | -2118.32 | -2099.94 | -2110.44 | -0.037 | 1.29 | 13.45 |
| 4×4×4 | -5021.28 | -5002.84 | -5013.03 | -0.005 | 0.99 | 13.44 |
| 5×5×5 | -9807.05 | -9788.61 | -9798.59 | -0.011 | 0.80 | 13.47 |

### 3.3 转变能级计算

**定义**
- 施主能级ε_D(+/0)：+1与中性态能量相等点
- 受主能级ε_A(0/-)：中性与-1态能量相等点

**计算方法**
```
ε(q/q') = [E^f_q(ε_F=0) - E^f_{q'}(ε_F=0)] / (q' - q)
```

## 5. 多参考与系综DFT

### 5.1 系综DFT (Ensemble DFT)

**理论基础**
- 对系综权重微分获得KS波函数
- 可获得多参考波函数
- 不破坏对称性
- 保持光谱特征(简并性)

**优势**
| 特性 | 系综DFT | 传统TDDFT |
|------|---------|-----------|
| 双激发态 | 可描述 | 标准泛函失效 |
| 电荷转移态 | 改进 | 长程修正需要 |
| 对称性 | 保持 | 可能破坏 |
| 激子效应 | 更好 | 需要BSE |

### 5.2 稳态DFT (i-DFT)

**Jacob等人发展**
- 从平衡DFT计算提取多体谱函数
- 相互作用多电子系统
- 非平衡稳态扩展

### 5.3 MC-srDFT方法

**长程MCSCF-短程DFT**
- 长程：多参考波函数方法
- 短程：DFT处理相关效应
- 激发能与跃迁性质计算
- 精度优于传统TDDFT

### 5.4 自旋翻转方法总结

**SF-CIS/TDDFT**
```
适用条件：
  - UHF起始波函数
  - 多重度≥3
  - 开壳层体系

计算类型：
  α电子→β轨道跃迁
  M_S = M_S(UHF) - 1

注意事项：
  - 自旋污染监控
  - 纯DFT需足够HF交换
  - 新基态为SF态1
```

### A. 缺陷形成能计算VASP脚本

```bash
#!/bin/bash
# 缺陷计算自动化脚本

# 完美超胞
mkdir perfect; cd perfect
cat > INCAR << EOF
ENCUT = 520
ISMEAR = 0; SIGMA = 0.05
NELM = 100
EDIFF = 1E-6
EOF
cp ../POSCAR_perfect ./POSCAR
vasp_std
cd ..

# 中性缺陷
mkdir defect_0; cd defect_0
cp ../INCAR ./
# 编辑POSCAR移除/添加原子
vasp_std
cd ..

# 带电缺陷(-2)
mkdir defect_q; cd defect_q
cp ../INCAR ./
cat >> INCAR << EOF
NELECT = [调整电子数]
EOF
vasp_std
cd ..

# 计算形成能
python << PYEOF
import numpy as np

E_perfect = float(open('perfect/OUTCAR').read().split('energy  without')[1].split()[0])
E_defect_0 = float(open('defect_0/OUTCAR').read().split('energy  without')[1].split()[0])
E_defect_q = float(open('defect_q/OUTCAR').read().split('energy  without')[1].split()[0])

# 化学势和修正项需根据实际情况设置
mu = 0  # 原子化学势
E_corr = 0  # 有限尺寸修正
epsilon_F = 0  # 费米能级
DeltaV = 0  # 电势对齐

Ef_0 = E_defect_0 - E_perfect + mu
Ef_q = E_defect_q - E_perfect + E_corr + mu + (-2)*(epsilon_F + DeltaV)

print(f"Neutral formation energy: {Ef_0:.3f} eV")
print(f"Charged formation energy: {Ef_q:.3f} eV")
PYEOF
```

### B. RT-TDDFT分析脚本

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.fft import fft, fftfreq

# 读取时间依赖偶极矩
t, dipole = np.loadtxt('dipole.dat', unpack=True)
dt = t[1] - t[0]

# 傅里叶变换获得光谱
N = len(dipole)
yf = fft(dipole)
xf = fftfreq(N, dt)[:N//2]

# 计算极化率与吸收谱
alpha = -yf[:N//2] / (xf + 1j*1e-8)**2
absorption = np.imag(alpha)

plt.plot(xf*27.2114, absorption)  # 转换为eV
plt.xlabel('Energy (eV)')
plt.ylabel('Absorption')
plt.xlim(0, 50)
plt.show()
```
