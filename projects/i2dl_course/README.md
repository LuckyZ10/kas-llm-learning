## 完整学习路径 (12讲)

### 第一阶段：基础篇 (预计 2-3 周)
- [ ] **Lecture 2**: 机器学习基础
  - 图像分类任务
  - 线性回归、逻辑回归
  - 损失函数、最大似然估计
  - 训练集/验证集/测试集划分
  - 过拟合与正则化

### 第二阶段：核心篇 (预计 3-4 周)
- [ ] **Lecture 3**: 神经网络入门
  - 从线性回归到神经网络
  - 激活函数
  - 多层感知机 (MLP)
  - 计算图

- [ ] **Lecture 4**: 优化与反向传播
  - 梯度下降
  - 反向传播算法
  - 链式法则
  - 参数初始化

- [ ] **Lecture 5**: 大规模优化
  - 随机梯度下降 (SGD)
  - Mini-batch
  - 学习率调度
  - 动量法

### 第三阶段：进阶篇 (预计 3-4 周)
- [ ] **Lecture 6**: 训练神经网络 I
  - 训练技巧
  - 批量归一化 (Batch Normalization)
  
- [ ] **Lecture 7**: 损失函数与激活函数
  - 各种损失函数详解
  - 激活函数选择与比较

- [ ] **Lecture 8**: 数据增强与正则化
  - 数据增强技术
  - Dropout、L2正则化等

### 第四阶段：CNN专题 (预计 2-3 周)
- [ ] **Lecture 9**: 卷积神经网络
  - 卷积操作
  - 池化层
  - CNN结构

- [ ] **Lecture 10**: CNN架构
  - LeNet、AlexNet、VGG、ResNet等经典架构
  - 架构设计原则

### 第五阶段：序列与高级主题 (预计 2-3 周)
- [ ] **Lecture 11**: RNN与Transformer
  - 循环神经网络 (RNN)
  - LSTM、GRU
  - Transformer架构
  - 注意力机制

- [ ] **Lecture 12**: 高级深度学习主题
  - 生成模型
  - 自监督学习
  - 前沿研究方向
