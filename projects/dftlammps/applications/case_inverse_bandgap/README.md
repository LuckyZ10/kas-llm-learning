# 目标带隙材料逆向设计案例

## 科学背景

### 带隙工程
带隙是半导体材料的核心参数，决定了其光电特性：
- **光伏材料**: 带隙约 1.0-1.6 eV 最优 (Shockley-Queisser极限)
- **LED**: 不同带隙对应不同发光波长
- **红外探测**: 窄带隙材料 (0.1-1.0 eV)
- **透明导电**: 宽带隙 (>3 eV) 保持可见光透明

### 二维材料带隙工程
二维过渡金属硫族化合物 (TMDC) 如 MoS₂、WSe₂ 等，具有层数依赖的带隙：
- 单层：直接带隙 (1.5-2.0 eV)
- 多层/体相：间接带隙 (1.2-1.3 eV)

调控手段：
1. **合金化**: Mo₁₋ₓWₓS₂ 合金连续调节带隙
2. **应变工程**: 双轴应变可改变带隙 ±0.5 eV/%
3. **缺陷工程**: 空位/掺杂引入带内态

## 方法概述

### 可微分DFT
本案例使用 JAX 自动微分实现从带隙到结构参数的梯度传播：
```
∂(Bandgap)/∂(Strain) → 最优应变值
∂(Bandgap)/∂(Composition) → 最优合金成分
```

### 代理模型
- **紧束缚模型**: d-p 轨道耦合快速估算带隙
- **神经网络**: 预训练模型提供快速预测和梯度

## 案例说明

### 目标
设计带隙为 1.5 eV 的 TMDC 材料，用于：
- 近红外光电探测
- 与硅光伏的界面匹配

### 初始材料
MoS₂ 单层 (带隙 ~1.8 eV)

### 优化变量
1. 金属层合金成分 (Mo/W 比例)
2. 硫族层合金成分 (S/Se/Te 比例)
3. 双轴应变大小

### 损失函数
```
L = (E_gap - E_target)² + λ₁|composition|² + λ₂|strain|²
```

python case_inverse_bandgap.py
