## 一、EOM-CCSD最新进展

### 1.1 理论基础

运动方程耦合簇方法（EOM-CC）将激发态表示为：
```
|Ψ_n⟩ = Σ_μ c_μ τ̂_μ |CC⟩ = exp(T̂) Σ_μ c_μ τ̂_μ |HF⟩
```

通过相似变换哈密顿量的对角化获得激发能：
```
E(c,c̄) = ⟨c̄|exp(-T̂)Ĥexp(T̂)|c⟩ / ⟨c̄|c⟩
```

### 1.2 EOM-CCSDT实现（2024-2025）

**Q-Chem最新更新**：

| 方法 | 标度 | 单激发精度 | 双激发精度 |
|------|------|-----------|-----------|
| **EOM-CCSD** | N⁶ | ~0.1-0.2 eV | >1 eV |
| **EOM-CCSDT** | N⁸ | ~0.01 eV | 0.1-0.2 eV |

**特点**：
- 严格大小广延性
- 支持SF/EE/IP/EA/DIP/DEA变体
- CCMAN2双精度/单精度实现
- 仅需点能计算

### 1.3 基组修正加速收敛（2024.04）

**巴黎索邦大学成果**：
- 密度基基组校正方法
- 线性响应理论框架
- EOM-CCSD激发能加速收敛
- aug-cc-pVTZ基组MAD降至0.02 eV

### 1.4 相对论核-价分离EOM-CCSD（2025）

**X2C-CVS-EOM-CCSD**：
- 精确二分量相对论实现
- 核激发态高效计算
- 重原子内壳层激发
- 系统包含相对论、关联、基组效应

### 1.5 EOM-CCSD-in-DFT嵌入（2024.05）

**单分子磁体应用**：
- Fe(II)/Fe(III)六水配合物
- Co(II)大单分子磁体
- CC2激发能误差<0.05 eV
- EOM-CCSD-in-DFT自旋-轨道耦合优势

## 二、ADC（代数图解构造）最新进展

### 2.1 理论基础

ADC源于极化传播子的图解微扰展开，通过中间态表示（ISR）构建厄米ADC矩阵：
```
M_IJ = ⟨Ψ̃_I^N|Ĥ - E_0^N|Ψ̃_J^N⟩
MX = XΩ
```

### 2.2 ADC方法层次

| 方法 | 标度 | 单激发 | 双激发 | 应用 |
|------|------|--------|--------|------|
| **ADC(2)-s** | N⁵ | 好 | 过高 | 大分子 |
| **ADC(2)-x** | N⁶ | 好 | 改善 | 诊断工具 |
| **ADC(3)** | N⁶ | 极好 | 好 | 定量预测 |

### 2.3 最新软件功能（Q-Chem）

**近期发展**：
- ADC(3)直接计算激发能、IP、EA
- 自旋翻转ADC处理多参考体系
- CVS-ADC(2/3)核激发态
- ADC(2/3)激发态梯度
- 环境模型（PCM/FDE）
- 跃迁密度矩阵分析

### 2.4 多参考ADC（MR-ADC）2024

**Fe(CO)₅瞬态X射线光电子谱模拟**：
- 核空穴屏蔽效应
- 自旋-轨道耦合
- 配体场分裂
- 与实验测量高度吻合

## 三、CASPT2最新进展

### 3.1 理论基础

完全活性空间微扰理论（CASPT2）以CASSCF波函数为零阶参考：
```
|Ψ^(0)⟩ = |Φ^CAS⟩
```

使用单电子（Fock-like）零阶哈密顿量，属于准简并微扰理论。

### 3.2 最新变体与发展

| 方法 | 特点 | 应用 |
|------|------|------|
| **RASPT2** | 限制活性空间 | 更大体系 |
| **CASPT3** | 三阶微扰 | IPEA位移独立性 |
| **CIPT2** | MRCI+CASPT2组合 | 高精度 |
| **CASPT2-F12** | 显式关联 | 基组极限 |
| **MS-CASPT2** | 多态混合 | 避免交叉 |
| **XMS-CASPT2** | 扩展多态混合 | 完全不变性 |

### 3.3 数值梯度实现（2024）

**MOLPRO实现**：
- 几何优化可用
- 解析梯度vs数值梯度验证
- 激发态势能面探索

### 3.4 瞬态吸收谱模拟（2024.06）

**WFOT工具发布**：
- 单参考/多参考波函数重叠
- TDDFT/CIS与CASSCF/CASPT2桥接
- 乙酰丙酮瞬态光谱模拟
- COBRAMM包接口

## 四、激发态方法对比

| 方法 | 参考态 | 双激发 | 计算成本 | 适用体系 |
|------|--------|--------|----------|----------|
| **TDDFT** | 单参考 | 无 | 低 | 大分子单激发 |
| **EOM-CCSD** | 单参考 | 近似 | 中高 | 中等分子 |
| **EOM-CCSDT** | 单参考 | 完全 | 高 | 高精度需求 |
| **ADC(2)** | 单参考 | 近似 | 中 | 大分子 |
| **ADC(3)** | 单参考 | 更好 | 中高 | 定量预测 |
| **CASPT2** | 多参考 | 完全 | 高 | 强关联/避免交叉 |
| **MR-ADC** | 多参考 | 完全 | 高 | 过渡金属/核激发 |
