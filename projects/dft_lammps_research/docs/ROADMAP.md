## 优先级矩阵

### 影响力 vs 工作量

```
高影响力
    │
    │  ● 稳定性增强      ● AI驱动优化
    │  ● 文档完善        ● 云平台部署
    │
    │  ● 新增DFT代码     ● 自主发现
    │  ● 扩展ML势        ● 闭环设计
    │
    └──────────────────────────────────
      低工作量                      高工作量
```

### 当前优先级（按顺序）

1. **P0 - 关键**: 稳定性、测试、文档
2. **P1 - 高**: FHI-aims、CHGNet、M3GNet
3. **P2 - 中**: 云平台、Web界面、GNN增强
4. **P3 - 低**: 自主发现、跨学科扩展
