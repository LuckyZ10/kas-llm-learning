# 机器学习力场前沿研究综述报告
## 模块5：力场基准测试与评估（Benchmarks & Evaluation）

### 5.1 基准测试的重要性

标准化基准测试是推动机器学习力场（MLIP）发展的核心驱动力：

**作用**:
- 公平比较不同方法
- 追踪领域进展
- 指导方法选择
- 发现研究空白

**类比**:
- ImageNet → 计算机视觉
- MMLU-Pro → 大语言模型
- CASP → 蛋白质结构预测

**基本信息**:
- 分子数: 130,831
- 元素: C, H, O, N, F
- 重原子数: ≤9
- 用途: 分子性质预测

**预测目标** (12项):
| 性质 | 单位 | SOTA方法 |
|------|------|---------|
| 偶极矩 | Debye | MGNN |
| 极化率 | a₀³ | MGNN |
| HOMO能量 | eV | MGNN |
| LUMO能量 | eV | MGNN |
| 能隙 | eV | MGNN |
| 电子空间范围 | a₀² | MGNN |
| ZPVE | eV | MGNN |
| U₀ (0K内能) | eV | MGNN |
| U (298K内能) | eV | MGNN |
| H (焓) | eV | MGNN |
| G (自由能) | eV | MGNN |
| 热容 | cal/mol/K | MGNN |

**最新SOTA** (MGNN, 2025):
- 7项性质达到SOTA
- 其他5项与SOTA相当

- 分子: 10个小有机分子
- 数据: DFT（PBE+TS）轨迹
- 用途: 分子动力学力场评估

**分子列表**:
- 乙醇 (Ethanol)
- 丙二醛 (Malonaldehyde)
- 萘 (Naphthalene)
- 对乙酰氨基酚 (Paracetamol)
- 水杨酸 (Salicylic acid)
- 甲苯 (Toluene)
- 尿嘧啶 (Uracil)
- 阿司匹林 (Aspirin)
- 苯 (Benzene)
- 异氰酸 (Isocyanic acid)

**力MAE进展** (meV/Å):

|------|------|-----------|
| 2018 | SchNet | ~30 |
| 2019 | PhysNet | ~20 |
| 2020 | DimeNet++ | ~15 |
| 2022 | NequIP | ~8 |

**用途**:

**专注领域**: 无机材料稳定性预测

**评估内容**:

- 材料稳定性判断
- 晶体结构优化

**排名** (2025年8月):
| 排名 | 模型 | MAE (eV/atom) |
|------|------|---------------|
| 1 | SevenNet-MF-ompa | **最低** |
| 2 | GRACE-2L-OAM | 低 |
| 3 | DPA-3.1-3M | 低 |
| 4 | Orb-v3 | 中等 |
| 5 | MatterSim-v1-5M | 中等 |

#### 5.2.5 OC20 (Open Catalyst 2020)

**规模**:
- 催化表面吸附能预测
- 1.26亿个DFT计算
- 多种吸附物-催化剂组合

**任务**:
- 吸附能预测
- 结构弛豫
- 力预测

**SOTA**:
- GemNet-OC: 力MAE 15-20 meV/Å
- EquiformerV2: 类似精度

#### 5.2.6 LAMBench (2025年最新)

**目标**: 大规模原子模型（LAM）通用基准

**创新**:
- 跨领域评估（分子+材料+催化）
- 区分力场任务和性质计算任务
- 强调实际应用可用性

**评估维度**:
| 维度 | 描述 |
|------|------|
| 通用性 | 跨体系类型表现 |
| 稳定性 | MD模拟能量守恒 |
| 准确性 | 能量/力预测误差 |
| 可迁移性 | 下游任务微调能力 |

**力场任务结果** (维度误差):
| 模型 | 分子 | 无机材料 | 催化 |
|------|------|---------|------|

| SevenNet-MF | 良好 | **最佳** | 中等 |

### 5.3 评估指标

#### 5.3.1 能量预测

**均方根误差（RMSE）**:
```
RMSE = √[Σ(E_pred - E_ref)² / N]
```

**平均绝对误差（MAE）**:
```
MAE = Σ|E_pred - E_ref| / N
```

**能量排序准确性**:
```
Ranking Accuracy = 正确排序对数 / 总对数
```

#### 5.3.2 力预测

**力MAE/RMSE**:
- 各向同性平均
- 按元素分解

**力余弦相似度**:
```
cos(θ) = (F_pred · F_ref) / (|F_pred| |F_ref|)
```

#### 5.3.3 MD稳定性

**稳定性测试** (MD17-ethanol标准):
- 500K温度
- 300ps轨迹
- 无物理禁止状态

**能量守恒**:
```
ΔE/E = (E_max - E_min) / E_avg
```

**温度保持**:
- 目标温度与实际温度偏差

### 5.4 综合基准研究

#### 5.4.1 Al-Cu-Zr与Si-O对比研究 (2025)

**测试方法**:
- GAP, HDNNP, MTP, ACE, NequIP, Allegro, MACE

**Al-Cu-Zr结果**:
| 方法 | 力MAE | 排名 |
|------|-------|------|
| MACE | **最低** | 1 |
| Allegro | **最低** | 1 |
| NequIP | 略高 | 3 |
| non-linear ACE | 接近 | 4 |

**Si-O结果**:
| 方法 | 力MAE | 排名 |
|------|-------|------|
| NequIP | **最低** | 1 |
| MACE | 接近 | 2 |
| Allegro | 接近 | 2 |

**结论**:
- 没有单一方法在所有体系上最优
- 元素类型影响模型选择

#### 5.4.2 计算效率对比

**单CPU速度** (10⁸原子，2500步):
| 方法 | 速度 (proc-s/atom/step) |
|------|------------------------|
| SNAP | 10⁻⁶ |
| MTP | 10⁻⁶ ~ 10⁻⁵ |
| ACE | 10⁻⁶ ~ 10⁻⁵ |
| HDNNP | 10⁻⁵ |
| DeepMD | 10⁻⁵ |
| NequIP | 10⁻⁴ ~ 10⁻³ |
| MACE | 10⁻⁴ ~ 10⁻³ |
| Allegro | 10⁻⁴ (GPU优化) |

**GPU加速**:
- DeepMD: ~40倍加速
- Allegro: 设计用于GPU扩展

#### 5.4.3 数据效率（学习曲线）

**关键发现**:
- 等变模型（NequIP/MACE/Allegro）数据效率最高
- 线性方法（线性ACE）数据效率低
- MTP在中等数据量下收敛

### 5.6 基准测试最佳实践

#### 5.6.1 测试集设计

**原则**:
1. 与训练集分离
2. 覆盖关键化学环境
3. 包含挑战性案例
4. 统计显著性（足够样本）

#### 5.6.2 误差分析

**分解误差来源**:
- 化学环境类型
- 元素组合
- 局部结构类型
- 能量/力范围

**可视化**:
- 抛物线图（parity plot）
- 误差分布
- 残差分析

#### 5.6.3 统计显著性

**重复实验**:
- 多次随机划分
- 不同初始化
- 报告均值±标准差

### 5.8 模块总结

**关键要点**:
1. **QM9/MD17**是小分子力场的标准基准
2. **Matbench Discovery**是材料稳定性评估的标准
3. **OC20**定义了催化领域SOTA
4. **LAMBench**代表了通用LAM评估的新方向

**实践建议**:
- 新模型应在多个基准上验证
- 报告训练集大小和计算成本
- 进行MD稳定性测试
- 分析误差来源和失败案例

**选择指南**:
| 任务 | 推荐基准 |
|------|---------|
| 小分子力场 | MD17, QM9 |
| 材料稳定性 | Matbench Discovery |
| 催化 | OC20 |
| 通用LAM | LAMBench |
| 长时程MD | 自定义稳定性测试 |
