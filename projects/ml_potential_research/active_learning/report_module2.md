# 机器学习力场前沿研究综述报告
## 模块2：主动学习策略优化（Active Learning Strategies）

### 2.1 主动学习在ML力场中的重要性

主动学习（Active Learning, AL）是构建高质量机器学习力场的核心策略。由于原子构型和组成空间的无限性，训练数据的充分覆盖至关重要。主动学习通过智能选择"信息量最大"的数据点进行DFT标记，显著减少计算成本。

**核心问题**:
- 外推区域（extrapolative regions）识别
- 稀有事件（rare events）捕获
- 不确定性量化（Uncertainty Quantification, UQ）

---

### 2.2 Query by Committee（委员会查询）

**原理**:
使用多个模型（委员会）的预测分歧来估计数据不确定性。标准差越大，不确定性越高。

**实现方式**:
```python
# 委员会方法伪代码
ensemble_predictions = [model_i.predict(structure) for model_i in committee]
mean_pred = np.mean(ensemble_predictions)
uncertainty = np.std(ensemble_predictions)  # UQ值

# 选择策略
if uncertainty > threshold:
    select_for_dft_labeling(structure)
```

**委员会构建策略**:
| 策略 | 描述 | 适用场景 |
|------|------|---------|
| 不同初始化 | 相同架构，不同随机种子 | 通用 |
| 不同训练集 | 相同架构，不同数据子集 | 大数据集 |
| 不同超参数 | 不同网络深度/宽度 | 探索性训练 |
| 部分权重调整 | 只调整输出层 | 计算效率优先 |

**MACE多头部委员会** (2025年最新):
- 多个输出模块连接到相同的原子环境描述符
- 基础模型固定，只训练输出头部
- 可将训练集压缩至原始大小的**5%**

### 2.3 不确定性量化方法

#### 2.3.1 基于Ensemble的UQ

**优点**:
- 实现简单
- 可传播到物理性质的预测

**局限性**:
- 计算成本高（需运行多个模型）
- 对错误指定模型（misspecified model）敏感

#### 2.3.2 梯度不确定性（Gradient-based UQ）

**最新进展** (Zaverkin et al., 2024):
- 使用校准的梯度不确定性
- 计算成本低于Ensemble方法
- 可达到与Ensemble相当或更好的精度

**核心思想**:
```
利用自动微分计算不确定性
不确定性偏置MD可同时捕获外推区域和稀有事件
```

#### 2.3.3 高斯过程（GP）与t过程（TP）

**高斯过程回归（GPR）**:
- 通过后验方差提供内在不确定性度量
- 适用于主动学习但难以传播不确定性

**学生t过程（Student-t Process）**:
- GP的推广，增加自由度参数ν
- 更重的尾部，对异常值更鲁棒
- 在相同样本量下不确定性估计更可靠

**性能对比** (2023年研究):
| 方法 | 校准性 | 计算成本 | 小样本表现 |
|------|--------|---------|-----------|
| GP | 中等 | 高 | 良好 |
| TP | **优秀** | 高 | **优秀** |
| Ensemble | 良好 | 很高 | 良好 |
| 梯度UQ | 良好 | **低** | 中等 |

#### 2.3.4 后验校准（Post-hoc Calibration）

**CRUDE方法** (Calibrating Regression Uncertainty Distributions Empirically):
- 训练后改进不确定性量化
- 需要额外的校准数据集
- 结合TP使用效果更佳

### 2.4 主动学习工作流设计

#### 2.4.1 标准AL循环

```
┌─────────────────────────────────────────────────────────┐
│  1. 初始训练: 在小数据集上训练ML势                        │
│                      ↓                                  │
│  2. 探索采样: 运行MD/增强采样生成候选构型                  │
│                      ↓                                  │
│  3. 不确定性评估: 使用UQ方法评估每个构型                   │
│                      ↓                                  │
│  4. 数据选择: 选择高不确定性构型                         │
│                      ↓                                  │
│  5. DFT标记: 对选中构型进行DFT计算                       │
│                      ↓                                  │
│  6. 重训练: 合并新数据重训练模型                          │
│                      ↓                                  │
│  7. 收敛检查: 如未收敛返回步骤2                           │
└─────────────────────────────────────────────────────────┘
```

#### 2.4.2 增强采样结合AL

**Steered MD + AL** (2024):
- 使用导向分子动力学（SMD）探索反应路径
- 主动学习捕获化学键断裂/形成
- 适用于有机分子构象搜索

**不确定性偏置MD** (2024):
```
F_bias = F_MLIP + λ∇UQ

其中UQ是不确定性，λ是偏置强度
```

**优势**:
- 同时捕获外推区域和稀有事件
- 自动探索构型空间

### 2.5 数据增强与相似性检查

#### 2.5.1 数据增强策略

**随机噪声增强**:
- 对原子位置添加小幅度随机扰动
- 增加训练数据多样性

**相似性检查**:
- 避免添加与现有数据集过于相似的构型
- 使用构型空间距离度量（如RMSD）

#### 2.5.2 智能选择准则

**自适应阈值** (2024年改进):
```python
# 不使用绝对UQ阈值
UQ_mean = np.mean(UQ_values[:1000])
UQ_std = np.std(UQ_values[:1000])
threshold = UQ_mean + 3 * UQ_std  # 动态阈值
```

### 2.6 主动学习中的收敛判据

#### 2.6.1 基于不确定性的判据

```
收敛条件:
1. 平均UQ < 预设阈值
2. UQ分布稳定（迭代间变化 < 5%）
3. 新增数据对模型改进 < 阈值
```

#### 2.6.2 基于物理性质的判据

```
收敛条件:
1. MD轨迹能量守恒
2. 预测性质（如扩散系数）稳定
3. 与参考DFT计算差异 < 阈值
```

### 2.9 挑战与未来方向

#### 2.9.1 当前挑战

1. **计算成本**: Ensemble方法需要多模型推理
2. **收敛判据**: 缺乏普适的收敛标准
3. **复杂体系**: 多相/多尺度体系的AL策略

#### 2.9.2 2025年趋势

1. **轻量级UQ**: 梯度不确定性、单模型UQ方法
2. **自适应采样**: 智能探索策略（强化学习驱动）
3. **基础模型+AL**: 在大预训练模型上进行高效微调
4. **实验数据整合**: 将实验约束纳入AL循环

---

### 2.10 模块总结

**关键要点**:
1. **Query by Committee**仍是最可靠的UQ方法
2. **梯度不确定性**提供计算高效的替代方案
3. **不确定性偏置MD**可同时捕获外推区域和稀有事件
4. **自适应阈值**优于固定阈值选择策略

**实践建议**:
- 小体系/高精度: 使用Ensemble方法
- 大体系/高效率: 使用梯度UQ
- 反应/相变: 结合增强采样
- 基础模型: 固定主体，AL微调头部
