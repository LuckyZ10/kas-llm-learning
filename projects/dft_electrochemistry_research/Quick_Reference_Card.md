## 关键方程

**CHE模型自由能**:
```
ΔG = ΔE_DFT + ΔZPE - TΔS - neU + kT·ln(10)·pH
```

**极限电位**:
```
U_L = -ΔG_max / e
η = U_eq - U_L
```

**火山图描述符**:
- HER: ΔG_H* (最优 ≈ 0 eV)
- OER/ORR: ΔG_O* - ΔG_OH* (最优 ≈ 1.97 eV)

## 常见陷阱

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| 忽略自旋极化 | 开放壳层体系必须ISPIN=2 |
| 固定晶格优化吸附物 | 固定底层原子，弛豫表面层 |
| 忽略ZPE | 必须计算吸附物频率 |
| 直接比较不同泛函 | 同系列比较需用相同泛函 |
| 忽略溶剂化 | 极性吸附物需加溶剂校正 |

## 收敛检查清单

- [ ] 力收敛 < 0.02 eV/Å
- [ ] 能量收敛 < 1E-5 eV
- [ ] 频率无虚频 (除TS外)
- [ ] k点收敛测试
- [ ] 截断能收敛测试
- [ ] 真空层 > 15 Å
