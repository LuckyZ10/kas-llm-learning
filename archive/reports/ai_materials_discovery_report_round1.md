## 模块1: 生成式AI材料设计 🧬

### 1.1 CDVAE (Crystal Diffusion Variational Autoencoder)

**核心贡献**: 2021年由Xie等人提出，开创了扩散模型在晶体生成领域的先河

**技术架构**:
- **编码器**: 周期性图神经网络(PGNN)将材料映射到潜空间
- **属性预测器**: 预测成分、晶格参数和原子数量
- **扩散解码器**: 通过朗之万动力学同时去噪原子坐标和更新原子类型
- **等变处理**: 使用SE(3)等变图神经网络保持置换、平移和旋转不变性

**最新进展 (2024-2025)**:
| 变体模型 | 核心改进 | 来源 |
|---------|---------|------|
| DP-CDVAE | 扩散概率模型替代分数匹配，能量误差降低68.1 meV/atom | Scientific Reports 2024 |
| LCOM | 引入替代保守模型最小化形成能 | arXiv 2023 |
| SyMat | 增强平移不变性 | NeurIPS 2024 |
| GemmsDiff | 允许晶格参数在扩散过程中演化 | AAAI 2024 |
| Con-CDVAE | 条件生成方法 | Comp. Mater. Today 2024 |

**应用案例**:
- 超导材料设计: 训练于1058个JARVIS结构，生成3000候选
- 2D材料生成: 从C2DB训练生成10,000候选，89%通过几何检查
- 锂离子电池材料: 发现528种潜在新导体，是之前研究的25倍

### 1.2 DiffCSP (Diffusion Crystal Structure Prediction)

**核心突破**: 2023年NeurIPS，首次实现联合等变扩散生成晶格和分数坐标

**技术创新**:
```
1. 周期性E(3)等变去噪模型
   - 同时生成晶格矩阵L和分数坐标F
   - 保持平移、旋转、周期性不变性

2. 分数坐标系统优势
   - 内建周期性编码
   - 使用Wrapped Normal分布建模周期性
   - 通过傅里叶变换促进去噪模型设计

3. 联合扩散过程
   - 损失函数: L_L/A = E[||ε - ε̂(M_t, t)||²]
   - 分数坐标使用分数匹配框架
```

**性能表现**:
- 在MP-20、Perov-5、Carbon-24等基准测试中超越现有学习方法
- 计算成本远低于DFT方法
- 支持固定成分CSP和原子类型生成

**扩展版本**:
| 版本 | 特性 | 年份 |
|-----|------|------|
| DiffCSP++ | 空间群约束生成 | 2024 |
| SCIGEN | 结构约束掩码策略 | 2024 |
| EquiCSP | 超图扩散 + 对称性增强 | 2025 |

### 1.3 CrystalLLM & FlowLLM

#### CrystalLLM (2024)
**创新点**: 基于LLaMA-2的自回归大语言模型，直接建模CIF文本格式

**工作流程**:
1. 数百万CIF文件训练
2. 蒙特卡洛树搜索整合形成能预测
3. 生成广泛无机化合物的合理晶体结构

**意义**: 挑战传统晶体表示方法，展示LLM学习晶体化学"世界模型"的潜力

#### FlowLLM (NeurIPS 2024) ⭐
**突破成果**:
- **稳定率提升300%** (相比先前最佳模型)
- **S.U.N.率提升50%** (Stable, Unique, Novel)

**架构设计**:
```
LLM (微调) → 基础分布学习 (文本表示)
    ↓
Riemannian Flow Matching (RFM)
    ↓
迭代精化坐标和晶格参数
```

**核心优势**:
- 弥合离散与连续建模差距
- 保留LLM自然语言提示能力
- 生成结构更接近弛豫态，降低后处理成本

**数据集**: MP-20 (45,231种材料)

## 模块2: 图神经网络材料性质预测 🕸️

### 2.1 ALIGNN (Atomistic Line Graph Neural Network)

**核心创新**: 使用线图显式编码键角信息，捕捉原子三元组角度关系

**架构特点**:
```python
# 顶点更新
v_i^(t+1) = v_i^(t) + SiLU(BN(W_s^(t) v_i^(t) + Σ_j u_(i,j)^(t) W_d^(t) v_j^(t)))

# 边更新 (原图和线图)
u_(i,j)^(t+1) = u_(i,j)^(t) + SiLU(BN(W_a^(t) v_i^(t-1) + W_b^(t) v_j^(t-1) + W_c^(t) u_(i,j)^(t-1)))
```

**性能基准** (Materials Project):
| 模型 | 形成能MAE (eV/atom) | 带隙MAE (eV) |
|-----|-------------------|-------------|
| CGCNN | 0.039 | 0.388 |
| MEGNet | 0.028 | 0.33 |
| ALIGNN | **0.022** | **0.218** |
| Matformer | 0.021 | - |

**应用领域**:
- 2D材料功函数预测 (MAE: 0.20 eV)
- 锂离子电池材料筛选
- JARVIS-DFT数据集多项性能预测

## 模块4: 实验-计算闭环 🔁

### 4.1 主动学习 (Active Learning)

**核心机制**:
```
循环流程:
初始化训练集 → 训练代理模型 → 预测不确定性
                              ↓
执行新实验 ← 选择不确定性高样本 ← 采集函数决策
      ↓
  更新训练集 → 迭代优化
```

**采集函数类型**:
| 策略 | 描述 |
|-----|------|
| 纯探索 | 优先选择不确定性高区域 |
| 纯利用 | 优先选择预测最优区域 |
| 期望改进(EI) | 平衡探索与利用 |
| Thompson采样 | 基于概率分布采样 |

**材料科学应用**:
- **A-Lab**: 合成失败后自动优化后续配方
- **GNoME**: 主动学习迭代，发现率从50%提升至80%

---

### 4.2 贝叶斯优化 (Bayesian Optimization)

**技术框架**:
```
高斯过程(GP)作为代理模型
    ↓
后验分布更新 (贝叶斯规则)
    ↓
采集函数选择下一实验点
    ↓
执行实验并更新数据
```

**突破案例**:

#### 案例1: 高熵合金成分优化
- **搜索空间**: ~200万种可能成分
- **结果**: 仅7次迭代筛选出35种高硬度合金
- **性能提升**: 83.3%优于训练样本最大值

#### 案例2: 波士顿大学3D打印平台
- **硬件**: 5台3D打印机 + 六轴机械臂 + 电子秤 + 万能试验机
- **流程**: 参数设计 → 自动合成 → 性能测试 → 贝叶斯优化
- **效率**: 100次实验 > 1800次网格搜索，**提升60倍**

#### 案例3: 德克萨斯农工大学高熵合金SFE优化
- **搜索空间**: 36,273种成分
- **探索比例**: 仅0.125% (~40次实验)
- **结果**: 找到最小层错能

**工具生态**:
| 工具 | 特性 | 机构 |
|-----|------|-----|
| BayBE | 化学编码、迁移学习、多目标 | 拜耳 |
| AM-ARES | 多目标贝叶斯优化 + 安全探索 | MIT |
| BoTorch | PyTorch贝叶斯优化库 | Meta |
