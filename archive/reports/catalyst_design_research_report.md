## 专题一：吸附能计算与线性标度关系

### 1.1 吸附能计算方法

#### DFT计算基础公式
```
E_ads = E_{total} - E_{slab} - E_{adsorbate}

ΔG = ΔE + ΔE_{ZPE} - TΔS
```

**计算参数标准**:
| 参数 | 典型值 | 说明 |
|------|--------|------|
| 赝势 | PAW | 投影缀加波 |
| 泛函 | GGA-PBE | 通用梯度近似 |
| 截断能 | 400-550 eV | 平面波基组 |
| K点 | 3×3×1 (slab) | Monkhorst-Pack |
| 真空层 | ≥15 Å | 避免周期性相互作用 |
| 力收敛 | 0.01-0.05 eV/Å | 结构优化 |

#### 吉布斯自由能修正
- **零点能(ZPE)**: 通过振动频率计算
- **熵变**: 包括平动、转动、振动贡献
- **标准态校正**: 气相分子从0.1 MPa校正到标准浓度

### 1.2 线性标度关系(Scaling Relations)

#### 核心发现
由Abild-Pedersen等人在2007年发现，不同中间体吸附能之间存在线性相关：

```
ΔE_{ZH_x} = γ·ΔE_Z + ξ
```

其中Z = C, N, O, S，x = 0, 1, 2, 3

#### 典型标度关系
| 反应 | 标度关系 | 斜率(γ) | 截距(ξ) |
|------|----------|---------|---------|
| ORR/OER | ΔG_{OOH} vs ΔG_{OH} | ~1.0 | 3.2±0.2 eV |
| CO2RR | ΔG_{COOH} vs ΔG_{CO} | ~0.8 | ~1.5 eV |
| HER | ΔG_{H*} vs 各种金属 | - | - |

### 1.3 打破标度关系的策略（2024-2025最新进展）

#### 1. 应变工程
- **原理**: 晶格膨胀/压缩改变d带宽度，差异化影响不同中间体
- **理论**: Peterson等人提出应变可打破传统标度关系
- **应用**: 使催化剂同时更强结合一种中间体、更弱结合另一种

#### 2. 单原子催化剂(SACs)
- **机制**: 双齿配位稳定特定中间体(如OOH*)
- **突破**: 破坏OH*-OOH*标度关系
- **代表**: M-N4位点催化剂

#### 3. 双原子位点(DACs)
- **层间错位工程**: 调控金属-金属间距(d_M-M)
- **金属依赖性**: 
  - 强关联体系(Mn/Fe/Co): d轨道向费米能级移动
  - 弱关联体系(Ni/Cu): 范德华力驱动结构扰动

#### 4. 质子供体引入
- **效果**: 选择性稳定特定中间体
- **案例**: 聚吡咯(pPy)存在下MTP催化剂的LSR断裂

## 专题二：火山曲线构建与活性预测

### 2.1 Sabatier原理

催化活性与吸附强度的关系呈火山形：
- **吸附太弱**: 反应物无法有效活化
- **吸附太强**: 产物难以脱附，位点中毒
- **最优状态**: 中等吸附强度，位于火山顶部

### 2.2 火山曲线构建方法论

#### 步骤1: DFT计算中间体吸附自由能
```
ΔG = E_{DFT} + E_{ZPE} - T·S - e·U
```

#### 步骤2: 确定电位限制步骤
```
U_L = min(ΔG_i)/e  (i = 1,2,3,4)
η = U - U_L  (过电位)
```

#### 步骤3: 建立标度关系
```
ΔG_{intermediate} = α·descriptor + β
```

#### 步骤4: 绘制火山曲线
- 横坐标: 描述符(如ΔG_{OH*})
- 纵坐标: 活性度量(如log(TOF), 1/η)

### 2.3 五大反应的描述符

| 反应 | 主要描述符 | 最优值 | 理论极限过电位 |
|------|-----------|--------|----------------|
| HER | ΔG_{H*} | ~0 eV | ~0 V |
| OER | ΔG_{O*} - ΔG_{OH*} | ~1.6 eV | ~0.37 V |
| ORR | ΔG_{OH*} | ~1.0 eV | ~0.40 V |
| CO2RR→CO | ΔG_{CO*} | 中等强度 | - |
| CO2RR→HCOOH | ΔG_{OCHO*} | 中等强度 | - |
| NRR | ΔG_{N*} 或 ΔG_{NNH*} | 因催化剂而异 | - |

### 2.4 微动力学建模

#### 能量跨度模型(Energy Span Model)
```
δE = max(T_i - I_j + ΔG_{ij})
TOF = (k_B T/h)·exp(-δE/k_B T)
```

- **TDI**: Turnover Determining Intermediate (决定周转的中间体)
- **TDTS**: Turnover Determining Transition State (决定周转的过渡态)

#### 覆盖率依赖效应
- 吸附物-吸附物相互作用显著影响反应动力学
- 高覆盖率下吸附能降低(覆盖效应)

### 2.5 最新进展

#### GC-DFT方法（2024-2025）
- **大正则密度泛函理论**: 考虑表面覆盖率和电位效应
- **恒电位计算**: 电子数可变，保持恒定费米能级
- **CIP-DFT**: 恒定内电势方法，适用于双电极体系

## 专题三：d带中心理论与电子结构分析

### 3.1 Hammer-Nørskov d带模型

#### 理论基础
- **提出**: Hammer & Nørskov (1995-2000)
- **核心**: d带中心位置决定吸附强度

#### 数学定义
```
ε_d = (∫ E·ρ_d(E)dE) / (∫ ρ_d(E)dE)
```

其中ρ_d(E)是d轨道的投影态密度(PDOS)

#### 物理图像
1. 吸附物与金属表面形成成键态和反键态
2. d带中心越接近费米能级：
   - 成键态填充更多
   - 反键态上移并空置
   - 吸附强度增强

### 3.2 吸附能预测模型

#### d带杂化贡献
```
E_{d-hyb} = -2(1-f)V²/|ε_d - ε_a| + 2(1+f)αV²
```

- **f**: d带填充因子
- **V**: 耦合矩阵元
- **ε_a**: 吸附物轨道能量

### 3.3 d带中心调控策略

| 策略 | 机制 | 效果 |
|------|------|------|
| **应变** | 晶格变化改变d带宽度 | 拉伸→εd上移；压缩→εd下移 |
| **合金化** | 电荷转移和轨道杂化 | d电子重新分布 |
| **配体效应** | 配体-金属相互作用 | 局域电子结构调节 |
| **尺寸效应** | 量子限域 | d带离散化 |

### 3.4 最新进展：BASED理论（2024）

#### 背景
- d带中心理论存在"异常现象"（反d带中心现象）
- 小金属颗粒d带不连续导致理论失效

#### BASED理论（Bonding and Anti-bonding orbitals Stable Electron Intensity Difference）
- **核心**: 键合与反键合轨道稳定电子强度差
- **准确率**: R² = 0.95
- **适用范围**: SACs、体相、不同吸附方式的体系

## 专题四：电化学界面模型

### 4.1 模型演进

| 模型 | 溶剂处理 | 电位处理 | 计算成本 | 适用场景 |
|------|----------|----------|----------|----------|
| **CHE** | 真空/隐式 | 后处理校正 | 低 | 高通量筛选 |
| **隐式溶剂** | 连续介质 | 恒电荷/恒电位 | 中 | 定性趋势 |
| **混合溶剂** | 显式+隐式 | 恒电位 | 中高 | 界面细节 |
| **AIMD** | 全显式 | 恒电位MD | 高 | 反应机理 |

### 4.2 计算氢电极(CHE)模型

#### 基本假设
```
H⁺ + e⁻ ↔ 1/2 H₂  (在U = 0 V vs RHE)
μ(H⁺) + μ(e⁻) = 1/2 μ(H₂) - eU
```

#### 自由能计算
```
ΔG(U) = ΔE_{DFT} + ΔZPE - TΔS - eU + ΔG_{pH}
```

### 4.3 超越CHE的模型

#### 恒电位方法
1. **GC-DFT**: 巨正则系综，电子化学势固定
2. **SJM**: 溶剂胶体模型，控制表面电荷
3. **CIP-DFT**: 恒定内电势，适用于双电极
4. **ESM-RISM**: 电子态模型+RISM，描述离子分布

#### 显式溶剂效应
- **氢键网络**: 水分子氢键稳定化中间体
- **质子转移**: Grotthuss机制参与反应
- **局部电场**: 双电层电场改变吸附能
- **pH效应**: 通过电位依赖的界面结构体现

### 4.4 最新进展（2024-2025）

#### CIP-DFT方法
- **创新**: 以内电势φ_M为基本变量
- **优势**: 计算开销仅比恒电荷高20-25%
- **应用**: 双电极体系、MD模拟

#### 机器学习势
- **目标**: 实现大规模恒电位模拟
- **挑战**: 准确描述电荷分布和长程相互作用
- **前景**: 融合量子力学精度与经典模拟效率

## 专题五：五大电催化反应专题

### 5.1 CO2还原反应(CO2RR)

#### 反应路径
```
CO路径:  CO₂ → *COOH → *CO → CO(g)
HCOOH路径: CO₂ → *OCHO → *HCOOH → HCOOH(aq)
多碳产物: 通过*CO-*CO偶联
```

#### 选择性控制描述符
| 描述符 | 物理意义 | 应用 |
|--------|----------|------|
| pKa([MP-H]⁰) | 金属氢化物酸度 | 预测CO vs HCOOH选择性 |
| ΔG_{CO*} | CO吸附能 | 多碳产物形成 |
| ΔG_{COOH*} - ΔG_{H*} | CO2RR vs HER竞争 | 选择性评估 |

#### 催化剂分类
| 类别 | 金属 | 主要产物 |
|------|------|----------|
| A | In, Pb | HCOOH |
| B | Ag, Au | CO |
| C | Ni, Pt | CO (但HER竞争) |
| D | Cu | 多碳产物(C₂H₄, C₂H₅OH) |

### 5.2 氮还原反应(NRR)

#### 反应机理
```
远端路径: N₂ → *NNH → *NNH₂ → *N + NH₃ → *NH₂ → NH₃
交替路径: N₂ → *NNH → *NHNH → *NHNH₂ → *NH + NH₃ → *NH₂ → NH₃
酶促路径: 类似固氮酶机制
```

#### 决速步
- **第一氢化**: N₂ → *NNH (通常电位限制)
- **最后氢化**: *NH₂ → NH₃

#### 单原子催化剂筛选结果
| 催化剂 | 极限电位 | 特点 |
|--------|----------|------|
| Mo@C₅N₂ | -0.39 V | 低过电位 |
| Os@SiCNT | 0.47 V | 高选择性 |
| Ru-TiNS | -0.40 V | 实验验证 |

#### 新型描述符
```
φ = (价电子数) / (电负性)
ICOHP: 晶体轨道哈密顿量积分
ΔG_N* - ΔG_H*: NRR vs HER选择性
```

### 5.3 析氢反应(HER)

#### 反应机理

**酸性介质**:
```
Volmer:   H⁺ + e⁻ + * → H*
Heyrovsky: H* + H⁺ + e⁻ → H₂
Tafel:     2H* → H₂
```

**碱性介质**:
```
Volmer:   H₂O + e⁻ + * → H* + OH⁻
Heyrovsky: H* + H₂O + e⁻ → H₂ + OH⁻
Tafel:     2H* → H₂
```

#### 活性描述符
- **关键**: ΔG_{H*} ≈ 0 eV
- **最佳催化剂**: Pt (ΔG_{H*} ≈ -0.09 eV)

#### 催化剂分类
| pH | 最优催化剂 | 过电位 |
|----|-----------|--------|
| 酸性 | Pt, Pt合金 | ~0 mV |
| 碱性 | Ni-Mo, Co-Ni | ~50-100 mV |
| 全pH | Pt/C, 单原子Pt | - |

### 5.4 析氧反应(OER)

#### 反应机理

**吸附演化机理(AEM)**:
```
* + H₂O → OH* + H⁺ + e⁻
OH* → O* + H⁺ + e⁻
O* + H₂O → OOH* + H⁺ + e⁻
OOH* → * + O₂ + H⁺ + e⁻
```

**晶格氧介导机理(LOM)**:
- 晶格氧直接参与反应
- 可能绕过AEM的标度关系限制
- 活性更高但稳定性可能降低

#### 标度关系限制
```
ΔG_{OOH*} = ΔG_{OH*} + 3.2 ± 0.2 eV
η_{min} = (3.2 - 2.46)/2 = 0.37 V
```

#### 最优催化剂
| 催化剂 | 过电位 | 机理 |
|--------|--------|------|
| IrO₂ | ~0.3 V | AEM |
| RuO₂ | ~0.35 V | AEM/LOM |
| NiFeOx | ~0.3 V | AEM |

### 5.5 氧还原反应(ORR)

#### 反应机理

**4e⁻路径**: O₂ + 4H⁺ + 4e⁻ → 2H₂O  (燃料电池)

**2e⁻路径**: O₂ + 2H⁺ + 2e⁻ → H₂O₂  (过氧化氢生产)

#### 标度关系限制
```
ΔG_{OOH*} = ΔG_{OH*} + 3.2 eV
η_{min} ≈ 0.4 V  (4e⁻ ORR)
```

#### 打破标度关系策略
| 策略 | 效果 | 代表催化剂 |
|------|------|-----------|
| 轴向配位 | 选择性稳定OOH* | FePc-O |
| 双原子位点 | 双齿吸附 | Fe₂-N₆ |
| 曲率效应 | 调控中间体几何 | 弯曲DACs |
| 应变工程 | 调节d带中心 | 应变Pt skin |
