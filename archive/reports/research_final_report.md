## 方向1：特定材料体系深度案例

### 1.1 半导体材料

#### 核心案例：III-Nitrides (BN, AlN, GaN)
**计算方法**：
- 软件：Abinit, WIEN2k, VASP
- 泛函：GGA-PBE
- 截断能：60 Ha
- 方法：Virtual Crystal Approximation (VCA)

**关键性质**：
| 性质 | 计算方法 | 精度 |
|------|---------|------|
| 晶格常数 | 结构优化 | <1%误差 |
| 带隙 | PBE0, HSE06, B3LYP | 接近实验值 |
| 光学性质 | DFPT | 介电函数、折射率 |

**最新进展**：
- β-(AlxGa1-x)2O3带隙工程：4.86 eV (x=0) → 5.80 eV (x=0.5)
- 应用：功率电子、HEMTs

#### 半导体带隙低估问题的解决方案
```
方法等级（按精度排序）：
LDA/GGA → metaGGA (SCAN) → HSE06 → PBE0 → GW
典型误差：
- PBE: 带隙低估30-50%
- HSE06: 误差<10%
- GW: 接近实验值（计算成本高10-100倍）
```

### 1.2 拓扑材料

#### 拓扑不变量计算
**Chern数计算方法**：
1. Berry曲率积分：C = (1/2π)∫Ω(k)d²k
2. Hybrid Wannier Charge Centers (HWCCs)

**Z2不变量计算**：
- **有反演对称性**：Fu-Kane奇偶性公式
  ```
  Z2 = Π(occupied) δi mod 2
  δi = Π eigenvalues at TRIM points
  ```
- **无反演对称性**：HWCC演化方法

**软件工具链**：
| 工具 | 功能 | 接口 |
|------|------|------|
| Z2Pack | Z2/Chern数 | VASP, QE, ABINIT |
| WannierTools | 拓扑表面态 | Wannier90 |
| BerryPI (WIEN2k) | Berry相 | 全电子 |
| WloopPHI | Weyl点 | WIEN2k |

**典型材料**：
- Bi2Se3：强拓扑绝缘体，Z2=(1;000)
- XSb4Te4 (X=Ge, Sn, Pb)：双重拓扑材料

### 1.4 催化剂材料

#### HER (析氢反应)
**计算氢电极模型 (CHE)**：
```
Volmer步骤: H+ + e- + * → H*
Heyrovsky步骤: H+ + e- + H* → H2 + *
Tafel步骤: H* + H* → H2 + 2*

ΔG_H* ≈ 0 eV 为最佳催化剂
 volcano曲线峰值附近
```

#### OER (析氧反应)
**4电子机理**：
```
* + H2O → OH* + H+ + e-
OH* → O* + H+ + e-
O* + H2O → OOH* + H+ + e-
OOH* → * + O2 + H+ + e-

理论过电位：η = max[ΔG]/e - 1.23 V
```

#### 单原子催化剂 (SACs)
**TM@TiTe2系统研究结果**：
- 筛选了27种过渡金属
- 14种稳定配置（不聚集）
- 关键描述符：
  - d带中心理论
  - BASED理论（新提出）
  - ICOHP

#### 溶剂效应处理
| 模型 | 特点 | 计算成本 |
|------|------|---------|
| 显式溶剂 | 包含水分子层，准确 | 高 |
| 隐式溶剂 (VASPsol等) | 连续介质近似，高效 | 低 |
| ELSI模型 | 固液界面显式处理 | 中等 |

## 方向2：跨尺度模拟方法

### 2.1 QM/MM方法

#### ONIOM方法框架
```
2层ONIOM:
E_total = E_high(model) + E_low(real) - E_low(model)

3层ONIOM:
E_total = E_QM1 + E_QM2 - E_QM1 + E_MM - E_QM2

典型设置：
- 高层：DFT (活性位点)
- 中层：DFTB/SE (邻近区域)
- 低层：MM力场 (环境)
```

#### 2025年最新进展
- **ML加速QM/MM**：使用ANI-1ccx, AIQM1等ML势替代DFT，速度提升1000倍
- **ONIOM(QM/QM')非绝热动力学**：用于光开关材料研究

### 2.2 DFT-MD耦合 (AIMD)

#### 方法对比
| 方法 | 特点 | 时间步长 | 适用体系 |
|------|------|---------|---------|
| BOMD | 每步DFT自洽 | ~1 fs | 数百原子 |
| CPMD | 扩展拉格朗日 | ~1 fs | 数百原子 |

#### 应用案例
- 锂离子电池SEI形成机理
- 高温氧化表面动力学
- 固液界面结构演化

**限制**：
- 体系规模：~500原子

- 计算成本：极高

#### 不同泛函精度对比
| 方法 | 典型误差 | 计算成本 |
|------|---------|---------|
| PBE | 低估30-50% | 1x |

| GW | <5% | 100-1000x |

#### 多保真度校正学习
```
Correction Learning方法：
1. 计算PBE带隙（低成本）
2. ML预测修正值：Δ = E_exp - E_PBE
3. 最终预测：E_pred = E_PBE + Δ_ML

效果：MAE降低17%
```

### 3.2 形成能计算精度

#### SCAN vs PBE
```
SCAN优势：
- 强键合化合物：更准确
- 晶格常数：比PBE准确
- 自相互作用误差：更小

SCAN劣势：
- 弱键合化合物（金属间）：不如PBE
- 磁性系统：磁矩预测问题
- 计算成本：比PBE高5倍
```

### 3.3 系统验证流程
```
1. 结构验证
   - 晶格常数vs实验（误差<2%）
   - 原子位置（<0.01 Å）

2. 电子性质验证
   - 带隙vs实验（考虑GW/HSE06）
   - 态密度特征

3. 能量验证
   - 形成能vs实验
   - 反应能垒vs实验

4. 动态性质验证
   - 声子谱
   - 扩散系数
```

## 方向4：实用计算技巧与最佳实践

### 4.1 收敛性测试标准

#### K点网格收敛
```
测试方法：
1. 从Γ点(1×1×1)开始
2. 逐步增加密度(2×2×2, 3×3×3...)
3. 绘制总能量vs k点密度曲线
4. 当能量变化<1 meV/atom时收敛

典型设置：
- 金属：高k点密度（KSPACING ~0.2 Å⁻¹）
- 半导体：中等密度（KSPACING ~0.3 Å⁻¹）
- 绝缘体：较低密度（KSPACING ~0.4 Å⁻¹）

Monkhorst-Pack vs Γ点：
- 多数情况使用Monkhorst-Pack
- 大胞或某些对称性使用Γ点
```

#### 截断能收敛
```
测试流程：
1. 从较低截断能开始（如200 eV）
2. 逐步增加（50 eV步长）
3. 当总能量变化<1 meV/atom时收敛

典型值：
- 软元素（C, N, O）：400-500 eV
- 硬元素（过渡金属）：500-600 eV
- 稀土元素：600+ eV

注意事项：
- 截断能影响力和应力计算
- 不同性质可能需要不同截断能
```

### 4.2 泛函选择指南

#### 通用选择策略
```
步骤1：确定材料类型
├─ 金属/导体 → LDA/GGA
├─ 半导体/绝缘体 → metaGGA/Hybrid
├─ 强关联体系 → DFT+U
└─ 范德华材料 → DFT-D3/D4

步骤2：确定计算目标
├─ 结构优化 → PBE/PBEsol
├─ 能带结构 → HSE06/PBE0
├─ 形成能 → PBE/SCAN
└─ 反应机理 → PBE+rVV10

步骤3：验证计算
├─ 与实验数据对比
├─ 文献数据对比
└─ 更高精度方法验证
```

```
第一阶段：高通量筛选
```
